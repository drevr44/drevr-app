# Drevr App - تصميم واجهة التطبيق

## نظام الألوان (Brand Colors)

| اللون | الكود | الاستخدام |
|-------|-------|-----------|
| أصفر ذهبي | #FFD700 | العناصر المميزة، الأيقونات |
| برتقالي | #FF9500 | الأزرار الرئيسية، التدرجات |
| أحمر | #FF3B30 | التنبيهات، حالة الإلغاء |
| أسود | #1A1A1A | النصوص، الخلفيات الداكنة |
| أبيض | #FFFFFF | الخلفيات الفاتحة |

## قائمة الشاشات

### 1. شاشة الترحيب (Splash)
- عرض الشعار مع تأثير fade-in
- خلفية بتدرج من الأصفر للبرتقالي
- انتقال تلقائي للشاشة الرئيسية

### 2. الشاشة الرئيسية (Home)
- شريط علوي بالشعار
- زر "حجز رحلة جديدة" بارز
- قائمة الحجوزات الأخيرة (إن وجدت)

### 3. شاشة إنشاء الحجز (Create Booking)
- حقل نقطة الانطلاق
- حقل نقطة الوصول
- اختيار نوع الرحلة (ذهاب فقط / ذهاب وعودة)
- اختيار نوع السيارة (اقتصادي / مريح / فاخر)
- زر تأكيد الحجز

### 4. شاشة تفاصيل الحجز (Booking Details)
- معلومات الحجز (المواقع، النوع)
- حالة الحجز (شارة ملونة)
- قائمة العروض من السائقين (بطاقات)
- زر اختيار العرض
- حالة "العروض مغلقة" بعد الاختيار

### 5. شاشة طلب تغيير السائق (Driver Change)
- حقل سبب التغيير
- زر إرسال الطلب
- رسالة تأكيد

## تدفق المستخدم الرئيسي

1. المستخدم يفتح التطبيق → شاشة الترحيب
2. ينتقل للرئيسية → يضغط "حجز جديد"
3. يملأ بيانات الرحلة → يضغط "تأكيد"
4. ينتظر العروض → يختار عرض مناسب
5. الحجز مؤكد → يمكنه طلب تغيير السائق إذا لزم

## مكونات واجهة المستخدم

### الأزرار
- **رئيسي**: خلفية برتقالية (#FF9500)، نص أبيض، زوايا مستديرة
- **ثانوي**: حدود برتقالية، خلفية شفافة
- **خطر**: خلفية حمراء (#FF3B30)

### البطاقات
- خلفية بيضاء/داكنة حسب الوضع
- ظل خفيف
- زوايا مستديرة (16px)
- padding داخلي (16px)

### شارات الحالة
| الحالة | اللون |
|--------|-------|
| PENDING | أصفر #FFD700 |
| CONFIRMED | أخضر #22C55E |
| ACTIVE | برتقالي #FF9500 |
| COMPLETED | أزرق #0a7ea4 |
| CANCELLED | أحمر #FF3B30 |

## دعم RTL
- التطبيق يدعم العربية كلغة أساسية
- اتجاه النص من اليمين لليسار
- الأيقونات تتكيف مع الاتجاه
