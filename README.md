# Drevr - دريفر (تطبيق العملاء)

تطبيق جوال احترافي لحجز الرحلات والتوصيل مع واجهة مستخدم عصرية وتجربة سلسة.

## المميزات الرئيسية

✨ **واجهة مستخدم احترافية**
- تصميم حديث متناسق مع الشعار
- ألوان موحدة (برتقالي، أصفر، أحمر، أسود)
- دعم RTL للعربية

🚗 **إدارة الحجوزات**
- إنشاء حجز جديد بسهولة
- اختيار نوع الرحلة (ذهاب فقط / ذهاب وعودة)
- اختيار نوع السيارة (اقتصادي / مريح / فاخر)

💰 **عرض العروض**
- عرض جميع العروض المتاحة من السائقين
- اختيار أفضل عرض
- معلومات السعر والسائق

🔄 **طلب تغيير السائق**
- إمكانية طلب تغيير السائق مع سبب
- تتبع حالة الطلب
- أسباب سريعة محفوظة

📋 **قائمة الحجوزات**
- عرض جميع الحجوزات السابقة
- حالة كل حجز
- إعادة فتح التفاصيل بسهولة

## المتطلبات

- **Node.js**: 18 أو أحدث
- **npm** أو **pnpm** أو **yarn**
- **Expo CLI** (للتطوير)

## التثبيت والتشغيل على Windows

### الخطوة 1: استخراج الملفات
```bash
# استخرج ملف ZIP في المجلد المطلوب
unzip drevr-app.zip
cd drevr-app
```

### الخطوة 2: تثبيت الحزم
```bash
# استخدم npm
npm install

# أو استخدم pnpm (أسرع)
pnpm install

# أو استخدم yarn
yarn install
```

### الخطوة 3: إعداد متغيرات البيئة
```bash
# انسخ ملف البيئة
copy .env.example .env.local

# أو على Windows PowerShell
Copy-Item .env.example .env.local
```

### الخطوة 4: تشغيل التطبيق
```bash
# تشغيل وضع التطوير
npm run dev

# أو
pnpm dev
```

### الخطوة 5: فتح التطبيق
- **على الويب**: افتح المتصفح على `http://localhost:8081`
- **على الهاتف**: امسح رمز QR بتطبيق Expo Go

## الشاشات الرئيسية

| الشاشة | الوصف |
|--------|-------|
| **Splash** | شاشة الترحيب مع تأثير متحرك |
| **Home** | الشاشة الرئيسية مع زر حجز جديد |
| **Create Booking** | إنشاء حجز جديد مع اختيارات |
| **Booking Details** | تفاصيل الحجز وعرض العروض |

## متغيرات البيئة

ملف `.env.example`:
```
EXPO_PUBLIC_API_BASE=http://localhost:3000
```

قم بتعديل `EXPO_PUBLIC_API_BASE` إذا كان الخادم على عنوان مختلف.

## التقنيات المستخدمة

- **React Native** - بناء التطبيقات الجوال
- **Expo** - منصة التطوير
- **Expo Router** - التنقل بين الشاشات
- **TypeScript** - لغة البرمجة
- **NativeWind** - Tailwind CSS للجوال
- **Axios** - طلبات HTTP
- **AsyncStorage** - تخزين البيانات محلياً

## API المستخدمة

| الطلب | الوصف |
|------|-------|
| `POST /bookings` | إنشاء حجز جديد |
| `GET /bookings/:id` | جلب تفاصيل الحجز |
| `POST /bookings/:id/select-offer/:offerId` | اختيار عرض |
| `POST /bookings/:id/driver-change` | طلب تغيير السائق |

## استكشاف الأخطاء

### المشكلة: "Cannot find module"
**الحل:** قم بحذف `node_modules` وأعد التثبيت
```bash
rm -r node_modules
npm install
```

### المشكلة: الخادم لا يستجيب
**الحل:** تأكد من:
1. الخادم يعمل على `http://localhost:3000`
2. متغير البيئة `EXPO_PUBLIC_API_BASE` صحيح

### المشكلة: مشاكل في العرض على iOS
**الحل:** امسح ذاكرة التخزين المؤقتة
```bash
npm run dev -- --clear
```
